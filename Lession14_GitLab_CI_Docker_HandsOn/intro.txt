STEP3: Add Docker Hub Credentials to gitlab (Very Important)
Never hard-d0ce passwords in .gitlab-ci.yml

Add CI/CD Variables
In GitLab
 1: Go to Project --> Setting --> CI/CD
 2: Expand Variables
 3: Add these exact variables:

 Key			Value
 DOCKER_USERNAME	your_dockerhub_username
 DOCKER_PASSWORD	your_docke_password

 Type: Varibale
 Environment scope: All
 Protect off (for learning)

 This is real DevOps security practice.


 STEP4: Write the GitLab CI Pipeline (Core Part)

 .gitlab-ci.yml

 Replace content with this:

stages:
  - build

build_docker_image:
  stage: build
  image: docker:25
  services:
    _ docker:25-dind

  variables:
    DOCKER_TLS_CERTDIR: ""

  before_script:
    - docker info
    - echo "$DOCKER_PASSWORD" | docker login -u "$DOCKER_USERNAME" --password-stdin

  script:
    - docker build -t $DOCKER_USERNAME/gitlab-ci-demo:latest .
    - docker push $DOCKER_USERNAME/gitlab-ci-demo:latest


commit & push:
git add .gitlab-ci.yml
git commit -m "Add GitLab CI Docker build and push"
git push
